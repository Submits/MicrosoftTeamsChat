Microsoft Teams Self-Hosting Tutorial

Tutorial on how to create a custom Microsoft Teams Chat database.

Fauna Setup

• Create a FaunaDB database with name 'Teams', and set the Region Group to 'Europe (EU)'
• Create a collection in the database with the name 'users'
• Create an index for the collection with the name 'user_by_name' and make the term 'data.username'
• Create an API key for your database in the 'Security' tab (keep this saved somewhere)

Code Setup

• Go to https://github.com/submits/MicrosoftTeamsChat
• Download the project as a .zip, then extract it
• In both the 'authHandler.js' and 'dashboard.js' files, update the 'secret' in the first few lines of code with the API key you generated
• Save the files, and the simply host them using your host of choice

Usage

• Once up and running, the signup/login page can be accessed through 'https://your.domain.here/auth'
• This page will be at the index directory, but it isn't needed and can be changed/removed
